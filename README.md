# Next Tutorial App

Template with minimal functional for sites with SSR.

https://nextjs.org/learn/

## About

* ReactJs
* NextJs
* SSR
* Fetch Data

## Requirements

NodeJS 7.6^


## Installation

```bash
npm install
```


## Running Dev Server

```bash
npm run dev
```

## Building and Running Production Server

```bash
npm run build
npm start
```
